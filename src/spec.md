# Specification

## Summary
**Goal:** Remove all 'Learn More' sections containing video embeds from every page of the renewable energy website.

**Planned changes:**
- Remove 'Learn More' sections with VideoEmbed components from Overview page
- Remove 'Learn More' sections with VideoEmbed components from Solar Heating Systems page
- Remove 'Learn More' sections with VideoEmbed components from Solar Cells/Photovoltaic page
- Remove 'Learn More' sections with VideoEmbed components from Hydropower page
- Remove 'Learn More' sections with VideoEmbed components from Wind Power page
- Remove 'Learn More' sections with VideoEmbed components from Geothermal Energy page
- Remove 'Learn More' sections with VideoEmbed components from Biomass page
- Remove 'Learn More' sections with VideoEmbed components from Biofuels page

**User-visible outcome:** Users will no longer see embedded videos in 'Learn More' sections across all pages. All other educational content including diagrams, fact boxes, tables, and cards will remain intact.
